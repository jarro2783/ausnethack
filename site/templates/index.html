Welcome to AusNethack, Australia's first NetHack server.

The Server

To play NetHack, simply run the following in your console: 

ssh play@ausnethack.com

You will be asked to create a username and password, and give us your email. We will never send you email except if you forget your password.

What is AusNethack?

AusNethack is Australia's first public Nethack server. There is already a great Nethack server at alt.org. Unfortunately for those of us on the other side of the world, it is hosted on the East coast of the USA, so the latency is a bit high. Hence, AusNethack was born to provide a fast server for the Oceanic region of the world. Building a complete working server is a big job, so this site is a little light on functionality. The aim is that the functionality that does exist is correct. If you find any bugs the please {% include "contact_us.html" %} or submit a pull request on our github repository. We are also experimenting with the sorts of game statistics that are useful, so {% include "contact_us.html" %} if you have any ideas or requests.
