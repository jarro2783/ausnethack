What is AusNethack?

AusNethack is Australia's first public Nethack server. There is already a great Nethack server at alt.org. Unfortunately it is hosted on the East coast of the USA, so the latency is a bit too high for the other side of the world. Hence, AusNethack was born to provide a fast server for the Oceanic region of the world. Building a complete working server is a big job, so this site is a little light on functionality. The aim is for everything that is here to be correct. If you find any bugs the please {% include "contact_us.html" %} or submit a pull request on our github repository. We are also experimenting with the sorts of game statistics that are useful, so {% include "contact_us.html" %} if you have any ideas or requests.
